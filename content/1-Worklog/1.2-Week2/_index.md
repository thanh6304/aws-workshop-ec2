* Understand the fundamental networking concepts on AWS and the role of each component within a VPC.

### Week 2 Achievements:

* Understood the basic AWS networking architecture, including Regions, Availability Zones (AZs), and Virtual Private Clouds (VPCs).
* Learned the differences between Public and Private Subnets, as well as the roles of Internet Gateway (IGW), NAT Gateway, and Route Tables.
* Understood how AWS networking components connect and work together.
* Practiced creating and configuring a VPC, Subnets, Route Tables, an Internet Gateway, and basic Security Groups on AWS.
