### Week 3 Objectives:

* Understand the basic concepts of Amazon EC2 on AWS.
* Learn how to choose the appropriate Instance Type, AMI, and EBS based on application requirements.
* Become familiar with remote connection methods to EC2 using SSH.

* Understood the core components of Amazon EC2, including Instance Types, AMIs, EBS, and Elastic IP.
